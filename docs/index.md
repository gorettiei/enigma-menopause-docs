# ENIGMA Menopause Transition & Beyond

**Neuroendocrinology Working Group — Analysis & Processing Guide**

*Version 1 — May 2026*

---

## Welcome

This documentation provides step-by-step instructions for sites contributing to the **ENIGMA Menopause Transition & Beyond Mega-Analysis** — a large-scale, multi-site study mapping brain structural changes across the menopause transition.

This guide covers:

- How to prepare and format your **demographic and clinical data**
- How to run **FreeSurfer preprocessing** following ENIGMA standards
- How to perform **quality control** of your outputs
- How to **share your data** with the analysis team

---

## Data Sharing Options

Participating sites may contribute at one of three levels:

| Option | Description |
|--------|-------------|
| **Option 1** | Existing FreeSurfer-derived measures (v7 or v8) |
| **Option 2** | Local processing using our standardised pipelines |
| **Option 3** | Raw structural scans (NIFTI) for centralised processing |

All contributions must include phenotypic data — see [Getting Started](onboarding/getting-started.md).
